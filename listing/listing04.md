Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Выведутся цифры от 0 до 9, а потом программа уйдет в deadlock

Программа создает канал ch и запускает анонимную горутину, которая пишет числа от 0 до 9 в этот канал. 
Затем основная горутина читает из канала и выводит считанные значения. 
Однако, поскольку нет явного закрытия канала, программа заблокируется в строке for n := range ch, 
ожидая дополнительных данных. Тем самым произойдет deadlock. Чтобы это пофиксить, необходимо
закрыть канал (использовать close(ch)), после записи данных в него.

```
